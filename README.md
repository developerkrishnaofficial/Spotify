# Spotify Web Player Clone

A front-end Spotify-inspired web application built using HTML and CSS. This project recreates the look and feel of the Spotify Web Player, including navigation, library section, music cards, and a fixed music player interface.

## Features

- Spotify-style user interface
- Home page (`index.html`)
- Search page (`search.html`)
- Sidebar navigation with Home and Search
- Your Library section
- Playlist and album cards
- Fixed bottom music player
- Responsive layout elements
- Font Awesome icons
- Google Fonts integration

## Tech Stack

- HTML5
- CSS3
- Font Awesome
- Google Fonts (Quicksand, Montserrat, Nunito Sans)

## How to Run

1. Download or clone the project.
2. Extract the project folder if required.
3. Open `index.html` in any modern web browser.
4. Use the navigation menu to switch between Home and Search pages.
